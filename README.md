# DirectoryPracticeTool

小学校中学年のディレクトリ操作初期段階の児童向けに作成されたディレクトリ操作練習用ミニゲーム集です。

## 概要

このツールは、小学3年生を対象にGUIベースのファイル・フォルダ操作を段階的に学習するための3つのミッションで構成されています。

## 動作環境

- Windows 10/11
- Microsoft Word（ミッション3で必要）
- タッチパネルまたはマウス

## ミッション構成

### ミッション1: おかたづけチャレンジ
**難易度:** ★☆☆  
**学習内容:** 
- 1階層のファイル移動
- ドラッグ&ドロップ操作
- フォルダとファイルの概念理解

散らかったファイル（9個）を正しいフォルダ（3個）に整理します。

### ミッション2: しりょう整理
**難易度:** ★★☆  
**学習内容:**
- 2階層のファイル移動
- ファイル名の読解
- 複雑なフォルダ構造の理解

学年と教科に応じてファイル（10個）を正しいフォルダに整理します。

### ミッション3: ファイルめいれんしゅう
**難易度:** ★★★  
**学習内容:**
- 新規ファイル作成
- ファイル名の命名規則の理解
- リネーム操作

以下の命名規則に従ってWordファイルを作成します。  
命名規則: `{出席番号}{児童氏名}【{教科}】{単元名}.docx`

## ダウンロード方法(github初心者向け)
方法1.画面上部緑色のcodeボタンを押してDownload ZIPを選択する。ダウンロードされたファイルを解凍する。
方法2.画面右側のReleaseボタンからダウンロードする。(推奨)

## 使い方

1. 各ミッションのbatファイルをダブルクリック
2. 自動生成されたフォルダとファイルを使って操作練習
3. 完了後、`こたえあわせ.vbs`で正誤確認
4. batファイルは実行後に自動削除されます

## 注意事項

- 各ミッションは独立しているため、どの順番でも実行可能です
- ただし、難易度順（1→2→3）での実施を推奨します
